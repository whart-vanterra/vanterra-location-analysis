# Location Intelligence v3.0 Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Rebuild the Vanterra location intelligence system with a correct v3.0 scoring model and organized Python pipeline, presented via a Next.js static site with interactive weight exploration.

**Architecture:** Python pipeline scripts in `pipeline/` fetch data from external APIs and Supabase, score cities using config-driven v3.0 model, and export static JSON. Next.js app in `app/` renders the report with client-side rescoring for weight experimentation. Supabase (`loc_*` tables) is the canonical data store.

**Tech Stack:** Python 3 (supabase-py, requests), Next.js 14+ (App Router), TypeScript, Tailwind CSS, Supabase PostgreSQL

**Spec:** `docs/superpowers/specs/2026-03-12-location-intelligence-v3-design.md`

```python
import os
import re
from dotenv import load_dotenv
from supabase import create_client

def get_supabase():
    """Initialize Supabase client from .env.local."""
    load_dotenv(".env.local")
    url = os.environ["NEXT_PUBLIC_SUPABASE_URL"]
    key = os.environ["SUPABASE_SERVICE_ROLE_KEY"]
    return create_client(url, key)

def normalize_city_key(city: str, state: str, return_display: bool = False):
    """
    Normalize city/state into a canonical city_key.
    Returns city_key string, or (city_key, display_name) if return_display=True.
    """
    display = city.strip()
    city_clean = display.lower()
    state_clean = state.strip().lower()

    # Strip suffixes
    city_clean = re.sub(r'\s+(city|town|cdp)$', '', city_clean)

    # Normalize abbreviations
    city_clean = re.sub(r'\bst\.?\s', 'saint ', city_clean)
    city_clean = re.sub(r'\bft\.?\s', 'fort ', city_clean)
    city_clean = re.sub(r'\bmt\.?\s', 'mount ', city_clean)

    key = f"{city_clean.strip()}|{state_clean.strip()}"

    if return_display:
        return key, display
    return key
```

```python
import math

def haversine(lat1, lng1, lat2, lng2):
    """Calculate distance in miles between two lat/lng points."""
    R = 3959  # Earth radius in miles
    dlat = math.radians(lat2 - lat1)
    dlng = math.radians(lng2 - lng1)
    a = math.sin(dlat/2)**2 + math.cos(math.radians(lat1)) * math.cos(math.radians(lat2)) * math.sin(dlng/2)**2
    return R * 2 * math.asin(math.sqrt(a))
```

### Task 29: Canonical score verification test

**Files:**
- Create: `app/src/lib/scoring.verify.test.ts`

**Prerequisite:** Tasks 7-9 must have run (scoring pipeline + export must have generated `app/src/data/`).

- [ ] **Step 1: Write canonical verification test**

Create `app/src/lib/scoring.verify.test.ts`:

```typescript
import { describe, it, expect } from 'vitest';
import { calcCompositeScore } from './scoring';
import type { ScoringConfig } from './types';

// These imports require generated data — only run after pipeline/export.py
import recommendations from '../data/recommendations.json';
import config from '../data/config.json';

const typedConfig = config as unknown as ScoringConfig;

describe('canonical score verification (TS vs Python parity)', () => {
  it('all scores match canonical within ±0.1 tolerance', () => {
    const brands = Object.keys(recommendations);
    let checked = 0;

    for (const brandId of brands) {
      const recs = (recommendations as any)[brandId];
      if (!Array.isArray(recs)) continue;

      const maxVol = Math.max(...recs.map((r: any) => r.search_vol_total || 0));
      const maxPop = Math.min(
        Math.max(...recs.map((r: any) => r.population || 0)),
        typedConfig.market_demand.population_cap
      );

      for (const rec of recs) {
        if (!rec.canonical_composite) continue;
        const tsScore = calcCompositeScore({
          brandVol: rec.search_vol_total, maxBrandVol: maxVol,
          population: rec.population, maxBrandPop: maxPop,
          ownerPct: rec.owner_occupied_pct, income: rec.median_household_income,
          yearBuilt: rec.median_year_built, compIndex: rec.competition_index,
          sameBrandDist: rec.same_brand_distance_mi, sisterBrands: rec.sister_brands_nearby,
          config: typedConfig,
        });
        expect(Math.abs(tsScore - rec.canonical_composite)).toBeLessThan(0.1);
        checked++;
      }
    }
    expect(checked).toBeGreaterThan(0);
  });
});
```
